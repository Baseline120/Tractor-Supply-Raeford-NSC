# Tractor Supply Raeford NC Construction Status

Public project-status HTML for the Tractor Supply Raeford, North Carolina project.

## Repository purpose
This repo holds the current one-page construction status sheet and dated archive copies for external sharing and weekly updates.

## Contents
- `index.html` — current published status page
- `archive/` — dated archive snapshots

## Current published page
After GitHub Pages is enabled, the live page is intended to publish from `index.html` on the `main` branch.

## Update workflow
1. Update the project data in `index.html`.
2. Save a dated copy in `archive/` when issuing a new revision.
3. Commit and push changes to refresh the public record.

## Notes
- This repo is project-specific to TSC Raeford, NC.
- Verify milestone dates before publishing schedule changes when source dates conflict.
